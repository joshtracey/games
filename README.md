# Pass-and-Play Go page

Static GitHub Pages side page.

## Install

1. Create a folder in your GitHub Pages repo, for example:
   `go/`
2. Put `index.html` inside that folder.
3. Commit and push.
4. Visit:
   `https://YOUR-USERNAME.github.io/YOUR-REPO/go/`

For a user/organization page repo named `YOUR-USERNAME.github.io`, visit:
`https://YOUR-USERNAME.github.io/go/`
